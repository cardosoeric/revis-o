Dicas de Consumo Consciente

Planejamento Financeiro

Estabeleça metas financeiras de curto, médio e longo prazo. Crie um orçamento mensal e acompanhe seus gastos regularmente.

Compras Inteligentes

Pesquise preços, compare produtos e evite compras por impulso. Priorize a qualidade e durabilidade dos produtos.

Redução de Desperdício

Adote hábitos que reduzam o consumo de energia e água. Planeje suas refeições para evitar desperdício de alimentos.

Controle de Gastos

Utilize aplicativos ou planilhas para registrar todas as suas despesas. Identifique categorias onde pode economizar.

Consumo Sustentável

Prefira produtos com menor impacto ambiental. Considere a economia circular: reparar, reutilizar e reciclar.

Educação Financeira

Invista em seu conhecimento sobre finanças. Leia livros, participe de cursos e acompanhe notícias econômicas.

Glossário de Termos Financeiros

Orçamento
Planejamento financeiro que estima receitas e despesas por um período determinado.
Renda
Total de recursos financeiros recebidos por uma pessoa ou família em determinado período.
Despesa Fixa
Gastos que se repetem mensalmente com valores relativamente constantes.
Despesa Variável
Gastos que variam de valor conforme o consumo ou necessidades mensais.
Poupança
Parte da renda que não é gasta e é reservada para necessidades futuras.
Investimento
Aplicação de recursos financeiros com expectativa de ganhos no futuro.
Juros
Remuneração cobrada pelo empréstimo de dinheiro ou rendimento de aplicações financeiras.
Inflação
Aumento generalizado e contínuo dos preços de bens e serviços na economia.
Consumo Consciente
Prática de consumir de forma responsável, considerando impactos sociais e ambientais.
Endividamento
Situação em que uma pessoa ou família possui dívidas a pagar.

Tutorial Git

Controle de Versão com Git

Para baixar o Git entre no link:

https://git-scm.com/downloads

Configuração Inicial

Após a instalação, sempre configurar as credenciais do Git:

Configuração

git config --global user.name "Seu nome"
Configura seu nome de usuário
git config --global user.email seu@email
Configura seu email

Comandos Básicos

git init
Inicializar uma pasta com o git
git add nome_do_arquivo
Adicionar arquivos para gerar uma versão
git commit -m "Mensagem"
Gerar uma versão com mensagem descritiva
git status
Verificar arquivos que foram alterados
git log
Verificar o log das modificações

Comandos Básicos do Linux/Unix

Navegação e Arquivos

cd nome_da_pasta
Entrar em uma pasta
ls nome_da_pasta
Listar o conteúdo de uma pasta
mkdir nome_da_pasta
Criar uma pasta

Manipulação de Arquivos

rm -r nome_da_pasta
Excluir uma pasta e seu conteúdo
mv nome_atual nome_novo
Renomear um arquivo
mv arquivo pasta/
Mover um arquivo de lugar
cp arquivo destino/
Copiar um arquivo

Dica Importante

O Git é essencial para o controle de versão de projetos. Sempre use commits descritivos e mantenha seu repositório organizado para facilitar o trabalho em equipe e o gerenciamento do projeto.
